# Cargo Empire

Build the largest colonization network possible — claim every reachable neutral planet fast.
- Prioritize Hauler production and cargo tech over combat ships in early game.
- Set colonist routes from every inhabited planet outward; never leave a route slot empty.
- Declare war only after owning more planets than any rival — use economic dominance to win.
- A planet you colonize cannot be taken without a fight; spread wide before enemies do.
